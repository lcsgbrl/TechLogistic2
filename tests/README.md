# Testes do Sistema

Esta pasta contém os testes responsáveis por validar o funcionamento do sistema de otimização logística.

## tsp.test.js
Responsável por testar o algoritmo TSP e verificar se a menor rota está sendo calculada corretamente.

## performance.test.js
Responsável pelos testes de desempenho do sistema com diferentes quantidades de setores.

## validacoes.test.js
Responsável por validar entradas inválidas, produtos duplicados e setores inexistentes.

## integracao.test.js
Responsável por testar a comunicação entre frontend e backend.
