# Algoritmo TSP

## O que é o TSP

O Problema do Travelling Salesman Problem (TSP) é um algoritmo utilizado para encontrar a menor rota possível passando por vários pontos sem repetir caminhos desnecessários.

## Como o sistema utiliza o TSP

O sistema calcula automaticamente a melhor rota para o robô coletar produtos dentro do armazém.

### Vizinho Mais Próximo
Escolhe o setor mais próximo do ponto atual.

### Força Bruta
Testa todas as combinações possíveis para encontrar a menor rota.
